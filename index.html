100 Days of Code

Follow my progress through the 100 Days of Code challenge here and @poetsec on Twitter.

Day 0 - July 25, 2021, Sunday

Today's Progress: Today is prep day. I've put together this blog and decided that I'll start the challenge by working on some tools from an infosec course I'm taking. Throughout the 100 days I will also be working through some courses from Free Code Camp along with a few other projects mixed in as well.
Thoughts: This should be a fun way to make some progress in Python and shell scripting, and a good way to keep myself motivated to sit down and code each day.

Day 1 - July 26, 2021, Friday

Today's Progress: I started off the first day by working on a network scanner in Python using Scapy. First I wrote a simple network scanner by importing Scapy and defining a scan function that calls scapy.arping to perform the network scan (see code below). After seeing how that works, I started writing my own custom network scanner in which the scan function is written out step by step instead of just using scapy.arping. The custom scanner is a work in progress and I will continue working on it tomorrow.
Thoughts: The first network scanner script is interesting and works, but writing out a custom scan function will let me tweak things to be exactly how I want them and provide more detail in a more organized way.

Due to some unforseen circumstances I was unable to continue for a few days, so I will be redoing day 1.

Day 1 - July 29, 2021, Thursday

Today's Progress: I finished the custom network scanner with a scan function that goes through the process step by step. I used dictionaries to organize the information to be displayed in an easy to read table. I also used Argparse to add a user input option to specify the IP or range the user wants to scan.
Thoughts: It was interesting to read through the Argparse documentation. Its similarities to Optparse makes it easy to use and understand.
